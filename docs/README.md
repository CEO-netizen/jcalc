# JCalc
![Java](https://img.shields.io/badge/Java-OpenJDK-brightgreen?logo=java&logoColor=white)
Welcome to JCalc, a simple Java-based calculator application designed for ease of use and versatility. Whether you need to perform basic arithmetic or more complex calculations, JCalc has got you covered.


JCalc features:
- basic arithmetic operations (addition, subtraction, multiplication, division)
- Graphing functions
- conversion between time units and numerical bases
- various others found in -H and -h

Latest version: 1.0.9
Development info:
- split into multiple classes for better organization (Time, GraphingCalculator, Main, ExpressionParser)
- UI: CLI, currently no GUI
- compiling from source: `javac src/*.java -d .` then run with `java Main [options]`
- precompiled JAR available in the releases section
- currently looking for contributors to help with features and bug fixes. E-Mail me at aafire@firemail.cc if interested.
